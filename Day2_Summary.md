# Day 2 - Exploring the Superstore Dataset

**Dataset Overview:**
- Rows: 9,800
- Columns: 18
- Time Period: ~6 years ago (but still useful for learning)

**Key Metrics:**
- Total Sales: $2,261,536.78
- Average Sale: $230.76
- Highest Sale: $22,638.48 (bulk order)
- Lowest Sale: $0.444 (single cheap item)

**Lesson Learned:**
- Never trust your eyes. Always use formulas (SUM, COUNT, MAX, MIN).
- Extreme values are not errors. Keep them, but flag them.
- Always create a Summary sheet for quick reference.
